These files constitute a workflow, made by Jean-Pierre BORG (Institut de Recherche en Cancérologie de Montpellier -- équipe : "Bioinformatique et Biologie des Systèmes du Cancer"), to process biological images.
Starting from a Whole Scale Image (WSI), this workflow may be used to process images (segmentation, annotation, prediction using ML methods),
to get a lot of measurements directly from the images or by posterior calculation, then perform a large number of statistical computations.
These software programs are written in Python and use CellProfiler modules for segmentation and measurements concerning cells.
